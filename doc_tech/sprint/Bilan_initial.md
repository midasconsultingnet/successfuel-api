# Sprint - Module 2 : Bilan initial

## Objectif
Développer les fonctionnalités permettant d'établir le bilan initial d'une station-service avant l'utilisation du système.

## Étapes du sprint
1. **Actifs**
   - Immobilisations
   - Stocks carburant & boutique
   - Créances clients
   - Banque & caisse
   - Analyse comparative des actifs

2. **Passifs**
   - Capital social
   - Réserves
   - Emprunts
   - Dettes fournisseurs
   - Analyse comparative des passifs

## Livrables
- Formulaire de saisie du bilan initial
- Interface de gestion des immobilisations
- Interface de gestion des stocks initiaux
- Interface de gestion des créances clients
- Interface de gestion des dettes fournisseurs
- Module d'analyse comparative

## Tests
- Tests de validation des montants saisis
- Tests de cohérence entre actifs et passifs
- Tests de calcul des totaux
- Tests d'export des données
