# Sprint - Module 5 : Gestion des ventes

## Objectif
Développer les fonctionnalités permettant de gérer toutes les ventes effectuées dans la station-service.

## Étapes du sprint
1. **Ventes carburant**
   - Processus complet de création
   - Détails des ventes
   - Saisie des index de pistolet
   - Paiement (cash/crédit/écart pompiste)
   - Mise à jour trésorerie
   - Mise à jour stock
   - Écritures comptables
   - Modification / annulation
   - Suivi des performances pompistes
   - Analyse de rendement
   - Suivi des écarts anormaux

2. **Ventes boutique**
   - Émission de tickets de caisse
   - Paiement
   - Arrêt de compte caissier
   - Analyse de performance des caissiers

3. **Ventes services**
   - Création
   - Paiement
   - Arrêt de compte caissier
   - Suivi de la productivité

## Livrables
- Interface de création des ventes carburant
- Interface de saisie des index de pistolet
- Interface de gestion des paiements
- Système de mise à jour de trésorerie
- Système de mise à jour de stock
- Système d'écritures comptables
- Interface de modification/annulation
- Module de suivi des performances pompistes
- Module d'analyse de rendement
- Module de suivi des écarts anormaux
- Interface de génération des tickets de caisse
- Interface d'arrêt de compte caissier
- Module de suivi de la productivité
- Module de gestion des ventes services
