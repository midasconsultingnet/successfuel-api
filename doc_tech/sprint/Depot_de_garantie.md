# Sprint - Section spéciale : Dépôt de garantie

## Objectif
Développer les fonctionnalités permettant de gérer les dépôts de garantie des clients.

## Étapes du sprint
1. **Gestion des dépôts**
   - Enregistrement (client, montant, paiement, écriture au passif)
   - Utilisation (paiement dette/vente/écart)
   - Remboursement
   - Historique complet + solde visible client
   - Suivi des contrats de fidélité

## Livrables
- Interface d'enregistrement des dépôts de garantie
- Module de gestion des utilisations de dépôts
- Interface de remboursement des dépôts
- Historique complet des transactions
- Affichage du solde visible client
- Module de suivi des contrats de fidélité

## Tests
- Tests de validation des montants
- Tests de gestion des utilisations
- Tests de remboursement
- Tests de mise à jour des soldes
- Tests d'historisation
- Tests de conformité comptable
