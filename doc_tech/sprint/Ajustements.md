# Sprint - Module 8 : Ajustements

## Objectif
Développer les fonctionnalités permettant de gérer les ajustements de stock et les réinitialisations d'index.

## Étapes du sprint
1. **Stock boutique**
   - Sortie des périmés/cassés
   - Analyse des causes de péremption

2. **Réinitialisation d'index de pistolets**
   - Réinitialisation sécurisée des index de pistolets
   - Suivi des validations hiérarchiques

## Livrables
- Interface de gestion des sorties de stock boutique (périmés/cassés)
- Module d'analyse des causes de péremption
- Interface de réinitialisation sécurisée des index de pistolets
- Système de validation hiérarchique
- Journalisation des actions critiques

## Tests
- Tests de validation des ajustements de stock
- Tests d'analyse des causes de péremption
- Tests de sécurité des réinitialisations d'index
- Tests de validation hiérarchique
- Tests de journalisation des actions
