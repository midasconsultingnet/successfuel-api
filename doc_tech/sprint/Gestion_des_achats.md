# Sprint - Module 4 : Gestion des achats

## Objectif
Développer les fonctionnalités permettant de gérer tous les approvisionnements de la station-service.

## Étapes du sprint
1. **Achats carburant - Processus complet**
   - Création de l'achat
   - Détail des produits/commandes
   - Validation
   - Paiement
   - Livraison + mesure avant/après + écart
   - Mouvement de stock
   - Écritures comptables
   - Mise à jour trésorerie & dette
   - Modification / annulation
   - Suivi des coûts logistique
   - Analyse de la qualité du carburant reçu
   - Évaluation des fournisseurs

2. **Achats boutique**
   - Création + affectation à une famille de produits
   - Gestion des produits
   - Bon de commande
   - Paiement, mouvement de stock, écritures comptables
   - Modification / annulation
   - Suivi des performances des produits

## Livrables
- Interface de création d'achats carburant
- Interface de gestion des détails d'achat
- Interface de validation des achats
- Interface de gestion des paiements
- Interface de gestion des livraisons
- Interface de mesure avant/après livraison
- Système de calcul des écarts
- Interface de mouvement de stock
- Système d'écritures comptables automatiques
- Interface de mise à jour de trésorerie
- Interface de mise à jour des dettes
- Système de modification/annulation
- Interface de suivi des coûts logistique
- Interface d'analyse de la qualité du carburant
- Interface d'évaluation des fournisseurs
- Interface de gestion des achats boutique
- Interface de création de bons de commande
